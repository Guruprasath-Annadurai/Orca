# Orca Terms of Service (Draft)

**Last updated:** see git history for this file.

## 1. What Orca is

Orca is an AI assistant built on open-weight language models, run locally
via Ollama by default. It provides chat, document question-answering,
code execution, and related features described in the product.

## 2. Known limitations — stated plainly, not buried

Orca's own model cards (generated by `orca train card`) document real,
current accuracy and safety scores for each model variant. As of this
document's drafting, evaluated variants have shown meaningful error rates
on factual/technical questions and imperfect resistance to adversarial
prompting. **Do not rely on Orca as the sole basis for decisions with
legal, medical, financial, or safety consequences.** Verify important
outputs independently.

## 3. Acceptable use

You may not use Orca to:
- Generate content involving sexual exploitation of minors, in any form.
- Generate instructions for creating weapons intended for mass casualties
  (biological, chemical, nuclear, radiological, or viable explosive devices).
- Attempt to circumvent the service's safety measures for the purpose of
  generating content otherwise prohibited by these terms.
- Violate applicable law using outputs generated by the service.

Orca's input moderation system (`orca/serve/moderation.py`) actively
screens for some of these categories before generation. This is a
heuristic, keyword/pattern-based system — documented in the code as "a
floor, not a ceiling" — and its limitations are described in the model
cards, not hidden from you.

## 4. What we do differently on self-harm content

If your message indicates you may be in emotional distress or crisis,
Orca does not refuse to respond. Instead, it is designed to respond with
care and include crisis resources (e.g., the 988 Suicide & Crisis Lifeline
in the US). This is a deliberate design choice, not an oversight — refusing
to engage with someone in crisis is not a safety practice we follow.

**Orca is not a crisis service, a mental health provider, or a substitute
for professional help.** If you are in crisis, please contact a crisis
line or emergency services directly.
